# Contributors

This project is built using open-source models and tools from various organizations. We gratefully acknowledge the following contributors:

## AI Models & Tools

- **Cursor** - AI-powered code editor and development environment
- **Gemini** - Google's multimodal AI model (used for various agent tasks)
- **Qwen Coder** - Alibaba Cloud's code generation model (used as Reviewer in High mode)
- **Nvidia Nemo** - NVIDIA's conversational AI framework (Nemotron models used as Planner)

## Additional Acknowledgments

- **Cognizant** - MAKER paper (arXiv:2511.09030) that inspired this architecture
- **llama.cpp** - Efficient LLM inference engine with Metal acceleration
- **Arize Phoenix** - Observability and evaluation platform
- **Continue.dev** - VS Code extension for AI-assisted development
