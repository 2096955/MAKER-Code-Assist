# Available SubAgents for Delegation

**Total**: 63 specialized agents across all engineering disciplines

---

## How to Use

Delegate tasks to any agent using:
```
Task(subagent_type="agent-name", prompt="task description")
```

## Quick Reference

### For New Features
1. `planning-agent` → Break down requirements
2. `architect` → Design solution
3. `code-generator` → Implement code
4. `test-automator` → Create tests
5. `quality-control-enforcer` → Final validation

### For Bugs
1. `error-detective` → Find root cause
2. `debugger` → Fix the issue
3. `test-automator` → Add regression tests

### For Performance
1. `performance-engineer` → Profile bottlenecks
2. `database-optimizer` → Optimize queries

### For Security
1. `security-auditor` → Audit vulnerabilities
2. `backend-security-coder` / `frontend-security-coder` → Implement fixes

### For Deployment
1. `deployment-engineer` → Set up CI/CD
2. `kubernetes-architect` → Design K8s architecture
3. `devops-engineer` → Execute deployment
